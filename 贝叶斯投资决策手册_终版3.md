# 贝叶斯投资决策手册

---

## 序言：为什么需要这本手册

> 为学日益，为道日损。损之又损，以至于无为。无为而无不为。
> ——《道德经》第四十八章

一位社会心理学家说过这样一段话：

> "那些经济学家竟然会假设人是'理性人'，简直太荒谬了。早在50年前，实验研究就表明：人总会坚持明显错误的决策而不承认错误；人总会在吃大亏之后急于捞回损失却亏得更多；人在预测错误之后只会找理由，死也不会承认预测错了；如果事实不符合自己的想法，人经常不是改变想法，而是歪曲信息，甚至主动拒绝信息。"

这不是在描述少数人，这是人类的默认操作系统。

**人的默认操作系统：**
```
情绪 → 行动 → 事后找理由
```

**这本手册要替换成的操作系统：**
```
先验（诚实的基础概率）→ 证据更新 → 行动 → 预设失效条件
```

替换极其困难——你要用后天学来的工具，对抗几万年进化出来的本能。但正因为难，它才有价值：市场的超额回报来自差异，如果所有人都被情绪驱动，能系统性克服情绪的人就拥有了真实的alpha来源。

**贝叶斯不是让你变成理性人，而是在关键时刻不被自己的本能出卖。**

---

### 底层操作系统：顺势

老子说的"势"不是预测未来，而是感知当下已经在发生、但尚未被普遍认知的力量方向，然后顺着它。

用贝叶斯语言：
```
势   = 已有高效力证据支持的概率分布方向
顺势 = 让判断和仓位与这个概率分布匹配
逆势 = 用主观意愿对抗已经形成的证据链
```

**追涨杀跌是顺势吗？不是。**
价格是势的影子，追影子永远追不上本体。完整解释见第二章。

## 第零章：凭什么参与——普通人的不对称优势

> 知人者智，自知者明。
> ——《道德经》第三十三章

机构有研究团队、信息渠道、量化模型。普通人照搬机构框架，是在用别人的武器打一场自己没有优势的战争。

**不是"如何追上机构"，而是"如何在机构无法触达的地方找到属于自己的势"。**

---

### 彼得林奇：生活即研究

> 你在超市、餐厅里观察到的消费行为，比任何分析师报告都早6-12个月。

这是贝叶斯意义上的高效力早期证据——难以伪造，且只有真实用户才能感知。

**普通人的三类天然优势：**

| 优势类型 | 你比机构早的原因 | 例子 |
|--------|-------------|------|
| 行业内部视角 | 你就在现场，不是在看报告 | 供应链松紧、客户是否真的买单 |
| 消费者直觉 | 涨价后你还在买 = 定价权验证 | 彼得林奇因老婆每天去买了星巴克 |
| 本地信息 | 机构要等财报，你已经看到了 | 品牌突然排队、工厂大量招人 |

**实操清单：**
- [ ] 这个产品我自己用过吗？周围人在重复使用吗？
- [ ] 这家公司做的事，我能用一句话说清楚吗？说不清楚 → 放弃
- [ ] 这个机会是生活中发现的，还是别人推荐来的？

---

### 冯柳弱者体系：守拙，等聪明钱犯大错

> 洼则盈。
> ——《道德经》第二十二章

冯柳假定自己在信息、研究、时间各方面都处于市场最差水平，能依靠的只有时间、赔率与常识。这不是谦虚，是极其精准的贝叶斯先验诚实。

**三个核心动作：**

**① 等风险充分呈现，再逆向介入**
大幅下跌后跌因路人皆知，你不需要深度研究，只需判断：这个已知的坏消息，是否被市场过度定价了？

**② 赔率优先，概率次之**
赔率高但概率不确定 → 小仓位分散潜伏；赔率高且概率也高 → 重点集中突破。
普通人最常犯的错是反过来：等"确定"了再买，但此时赔率已经很差。

**③ 守拙，等聪明钱犯大错**
等机构因利欲熏心、在大得失前丧失心智，给出严重错误定价——这才是普通人真正的猎场。

**普通人的三类猎场：**

| 猎场 | 条件 | 优势来源 | 认知系数 |
|------|------|--------|--------|
| 生活中发现的消费品（彼得林奇型） | 你是真实用户，周围人重复购买 | 比机构早6-12个月感知需求 | 1.2 |
| 高关注度大票因非基本面大跌（冯柳型） | 跌因路人皆知，属情绪而非基本面 | 只需判断定价是否过度 | 1.0 |
| 你所在行业的上下游公司 | 真实的内部视角 | 机构永远看不到的毛细血管 | 1.2 |
| 纯跟随市场共识 | 无独特信息 | 无 | 0.8 |
| 道听途说 | 信息来源不可靠 | 负优势 | 0.3 |

## 第一章：选什么——三层过滤器

> 为道日损，损之又损，以至于无为。
> ——《道德经》第四十八章

与其问"这个机会有多好"，先问"这个机会有什么致命缺陷"。失败的路径比成功的路径少，更容易穷举，误差更小。

> 天下皆知美之为美，斯恶已。众人皆知的机会，赔率已经消失。真正的势，在众人看见之前已经形成。

---

### 第一层：芒格大道至简（一票否决）

三个二元问题，任意一个"否" → 直接淘汰：

**① 好生意？** 护城河存不存在？五年后还能维持定价权吗？

**② 好管理层？** 历史上有没有过一次损害股东利益的行为？有 → 否决，不分析程度。

**③ 逻辑简单？** 需要每季度重新判断宏观、政策、竞争格局 → 否决。只持有"不需要聪明就能预测"的生意。

**Inversion补充：** 先花同等时间回答"什么情况下会永久亏损"，再建立正向判断。芒格说："我只想知道我会死在哪里，这样我就永远不去那里。"

---

### 第二层：巴菲特能力圈过滤

- [ ] 这家公司的生意，我真的看得懂吗？
- [ ] 我的判断和市场共识，差在哪里？

> 第二个问题答不上来——没有认知优势，仓位上限自动降低。没有差异的判断，不产生alpha。

---

### 第三层：BG天花板判断

**十年后，这家公司最大可能是什么样子？**
关注量级，不关注小数点。不问现在值多少钱，问天花板在哪里。

**早期信号扫描（出现2条以上才进入下一章）：**

| 信号 | 为什么有效 | 证据效力 |
|------|----------|--------|
| 竞争对手开始抄它或试图收购 | 大公司不会对付不威胁自己的对手 | 高 |
| 涨价后客户流失低于预期 | 定价权最直接的证明 | 高 |
| 用户在产品不完善时已高度依赖 | 习惯养成早于产品完善 | 高 |
| 卖方分析师无法归类，模型套不进去 | 主流看不懂 = 定价可能不充分 | 中 |
| 创始人持续把利润投回研发而非分红 | 资本分配行为比语言更诚实 | 中高 |

---

### 证据效力判断——通用工具

**快速四关（所有信息适用）：**

**① 反面情况下这个证据会出现吗？**
CEO说前景好——前景差时也这么说？会 → 效力接近零

**② 这个证据有多难伪造？**
进了财报的数据 > 管理层口头表述 > 分析师预测

**③ 这些证据是独立的，还是同源的？**
10个KOL引用同一份报告 = 本质是1条证据

**④ 我是主动找到的，还是它找上我的？**
主动搜索到的反驳 > 算法推送的支持

### 排他性证据识别——进阶工具（ABCDE）

排他性证据的数学定义：
```
P(证据出现 | 假设为真) >> P(证据出现 | 假设为假)
理想状态：P(证据 | 假设为假) ≈ 0
```

真正的排他性证据出现时，股价通常已反应完毕。因此真正有价值的问题是：**在它被市场广泛认知之前，如何识别"接近排他性"的证据？**

**ABCDE检验：**

| 维度 | 问题 | 排他性高的特征 |
|------|------|------------|
| **A**symmetric（不对称） | 假设为假时，这个证据出现的概率有多低？ | < 10% |
| **B**ehavior（行为） | 这是行为证据还是语言证据？ | 用真实资金/采购/雇佣表达 |
| **C**ost（成本） | 产生这个证据需要付出多大真实成本？ | 成本越高，效力越高 |
| **D**ecoupled（独立） | 这条证据和已知证据真正独立吗？ | 来源完全不同 |
| **E**arly（早期） | 有多少人现在意识到了它的含义？ | 极少数人感知到 |

**五条满足 → 接近排他性，大幅更新判断，仓位系数显著提高**
**三条满足 → 中等效力，正常更新**
**两条以下 → 低效力，微调判断，不触发仓位变化**

**现实中最接近排他性的五类证据：**

| 类型 | 为什么接近排他性 | 例子 |
|------|-------------|------|
| 竞争对手放弃行为 | 成本极高，假设为假时几乎不出现 | 传统车企停止纯电研发转买电池 |
| 客户锁定行为 | 切换成本极高 | 云厂商重建工具链深度绑定特定芯片 |
| 监管针对性行动 | 规模已到不得不管的程度 | 欧盟专门为谷歌立法 |
| 内部人非常规资金 | 大跌中自掏腰包买入，逆势成本最高 | 股价大跌时加速回购（非期权行权） |
| 不可逆临界点 | 用户习惯已形成，价格战无法攻破 | DAU/MAU > 0.6；支付渗透率 > 70% |

**同一证据为什么在不同人眼里权重不同？**

```
证据权重 = 证据本身的效力 × 观察者的先验
```

先验不同，后验必然不同——这是贝叶斯的数学必然，不是主观随意。
当一条证据让先验差异极大的人得出了相同结论，这条证据才接近真正的排他性。

## 第二章：买多少——周期 × 凯利

> 反者道之动，弱者道之用。
> ——《道德经》第四十章

物极必反，周期的本质是过度之后的修正。霍华德的"钟摆"和老子的"反"是同一个观察。

**周期决定整体水位，凯利决定单笔比例。**

---

### 第一步：霍华德周期定位

周期不是时间的函数，是情绪偏离均值的函数。你不需要预测"什么时候"，只需判断"现在偏离了多远"。

**底部信号清单（出现5条以上 → P(底部) > 70%）：**

| 信号 | 底部时的典型状态 | 证据效力 |
|------|--------------|--------|
| 估值 | PE/PB跌至历史10%分位以下 | 高 |
| 信贷条件 | 银行收紧贷款，信用利差走阔 | 高 |
| IPO市场 | 新股发行停滞，破发率高 | 高 |
| 聪明钱行为 | 内部人增持，大机构开始建仓 | 高 |
| 基金流向 | 主动基金净流出超过3个季度 | 中高 |
| 媒体叙事 | 财经封面全是"崩溃""寒冬" | 中 |
| 分析师行为 | 大量撤销覆盖，目标价持续下调 | 中 |

**复苏启动的先行信号（一叶知秋）：**

| 时间 | 信号 |
|------|------|
| 第一片叶（6-9个月领先） | 最悲观的分析师开始上调；流动性最差的资产止跌；信用利差收窄但股市未反应 |
| 第二片叶（3-6个月领先） | 成交量低位放大；坏消息不跌；内部人增持进财报 |
| 第三片叶（同步，已较晚） | GDP止跌；媒体报道"复苏"；分析师集体上调目标价 |

**散户情绪是周期的反向指标：**

```
P(底部 | 散户极度恐惧、集体杀跌) >> P(底部 | 市场平静)
P(顶部 | 散户极度亢奋、集体追涨) >> P(顶部 | 市场平静)
```

| 散户感受 | 散户行为 | 实际周期位置 | 聪明钱在做什么 |
|--------|--------|-----------|------------|
| 极度亢奋，错过太可惜 | 追涨买入 | 顶部区间 | 出货 |
| 极度恐惧，跌了赶快跑 | 杀跌卖出 | 底部区间 | 建仓 |
| 平静，觉得没机会 | 观望 | 周期中段 | 持仓等待 |

> 你自己的恐惧和贪婪，是最诚实的周期位置指示器。
> 知足者富——仓位保守不是胆小，是对认知边界的诚实。

**周期系数对照表：**

| 周期位置 | 判断依据 | 周期系数 |
|--------|--------|--------|
| 明确底部复苏启动 | 底部信号5条+ 且第一片叶已落 | 1.5 |
| 底部区间复苏未确认 | 底部信号出现，先行指标未动 | 1.2 |
| 周期中段，中性 | 无明显极端信号 | 1.0 |
| 过热迹象出现 | 情绪亢奋，估值偏高 | 0.6 |
| 明确顶部特征 | 顶部信号5条以上 | 0.3 |

---

### 第二步：凯利仓位

**公式：**
```
最终仓位 = 半凯利基准 × 确定性系数 × 安全边际系数 × 认知优势系数 × 周期系数
```

**半凯利基准：**
```
期望值 = p₁×r₁ + p₂×r₂ + p₃×r₃
半凯利基准 = 期望值 ÷ 最大下行幅度 ÷ 2
```

**四个系数：**

| 系数 | 高 | 中 | 低 |
|------|----|----|-----|
| 确定性（能看多远） | 护城河清晰，格局5年稳定 → 1.0 | 行业变化快，有颠覆风险 → 0.6 | 依赖宏观政策 → 0.3 |
| 安全边际（价格） | 显然低估 → 1.5 | 合理定价 → 1.0 | 估值偏贵 → 0.6 / 明显高估 → 0.3 |
| 认知优势（差异） | 深度研究，与市场有实质差异 → 1.2 | 普通了解 → 0.8 | 道听途说 → 0.3 |
| 周期系数 | 见上表 | | |

**宁德时代示例：**
```
半凯利基准 7% × 确定性0.7 × 安全边际0.6 × 认知优势0.8 × 周期1.2 ≈ 2.8%
```

好公司在贵的时候，贝叶斯给你的仓位就是这么保守。巴菲特重仓可口可乐，是四个系数同时拉满的结果。

**加仓触发器（预设，不靠感觉）：**

| 触发条件 | 仓位变化 |
|--------|--------|
| 估值回调至合理区间 | 约翻倍 |
| 颠覆性威胁被证伪 | 提升约30% |
| 完成深度独立研究 | 提升约50% |
| 三者同时发生 | 可达8-10%，巴菲特级重仓 |

## 第三章：什么时候走——拐点审计 + 熔断

> 为之于未有，治之于未乱。
> ——《道德经》第六十四章

在问题尚未显现时就开始处理。不等排他性证据，在第一片叶落下时行动。

---

### 第一节：拐点审计

**好公司 ≠ 好的等待资产。经营质量和持有EV必须拆开来看。**

| # | 每次"再等等"之前必问 | 你的答案 |
|---|------------------|--------|
| ① | 拐点在持有窗口内发生的概率？ | |
| ② | 如果发生，超预期的强度有多大？ | |
| ③ | 市场已经提前定价了多少？ | |
| ④ | 等待期间的成本？（时间+机会+波动） | |
| ⑤ | 如果没来，仓位会跌多少？ | |

**三个常见幻觉：**

幻觉一：p = p₁(拐点发生) × p₂(拐点足够强) × p₃(在时间窗口内)，三者相乘往往远小于直觉

幻觉二：你需要的不是"拐点来了"，而是"拐点强度 > 市场已隐含的拐点强度"

幻觉三：等待带负carry——资本占用 + 替代机会流失 + 波动税 + 叙事绑架，四个通道同时消耗

---

### 第二节：股价信号解读

```
冲击力 = 实际消息 − 市场已隐含的预期
```

| 现象 | 贝叶斯含义 | 信号强度 |
|------|----------|--------|
| 坏消息出来不跌 | 悲观预期已充分定价 | 中 |
| 好消息出来不涨 | 乐观预期已定价，利好出尽风险 | 中高 |
| 两者同时出现 | 双向定价饱和，反转概率大幅提升 | 高 |
| 大盘跌个股涨 | 主动资金对抗系统性压力 | 高 |
| 三者同时出现 | 三条独立证据指向同一方向 | 极高 |

杀跌方向完全对称——散户感知到"在跌"时，距离势真正衰减已过去9-18个月，坏消息早已充分定价。**追涨杀跌不是方向错，是时机系统性滞后一个周期。**

---

### 第三节：一叶知秋熔断

**熔断规则：**

| 失效信号出现数量 | 行动 |
|-------------|------|
| 0-1条 | 正常持有，继续观察 |
| 2-3条 | 减半仓位，重新审计逻辑 |
| 4-5条 | 不等排他性证据，清仓 |

**以存储/光模块为例：**

| # | 信号 | 领先时间 |
|---|------|--------|

## 附录三：半凯利公式

**第一步：设定三个场景**

| 场景 | 概率 | 预期回报 |
|------|------|--------|
| 超预期，估值提升 | p₁ | r₁ |
| 符合预期，估值不变 | p₂ | r₂ |
| 不及预期或估值压缩 | p₃ | r₃ |

**第二步：计算期望值**
```
期望值 E = p₁×r₁ + p₂×r₂ + p₃×r₃
```

**第三步：计算半凯利基准**
```
半凯利基准 = E ÷ |最大下行幅度| ÷ 2
```

**第四步：乘以四个修正系数**
```
最终仓位 = 半凯利基准 × 确定性系数 × 安全边际系数 × 认知优势系数 × 周期系数
```

**宁德示例（完整演算）：**
```
场景设定：
  超预期（p=0.3，r=+100%）
  符合预期（p=0.5，r=+30%）
  不及预期（p=0.2，r=-30%）

期望值 E = 0.3×100% + 0.5×30% + 0.2×(-30%) = 30%+15%-6% = 39%
最大下行 = 30%
半凯利基准 = 39% ÷ 30% ÷ 2 = 6.5% ≈ 7%

修正系数：
  确定性    0.7（护城河清晰，固态电池是已知威胁）
  安全边际  0.6（估值偏贵）
  认知优势  0.8（普通了解，无独特信息）
  周期系数  1.2（底部区间复苏未完全确认）

最终仓位 = 7% × 0.7 × 0.6 × 0.8 × 1.2 ≈ 2.8%
```

**系数速查表：**

| 系数 | 1.5 | 1.2 | 1.0 | 0.6 | 0.3 |
|------|-----|-----|-----|-----|-----|
| 确定性 | — | — | 护城河清晰 | 有颠覆风险 | 依赖宏观政策 |
| 安全边际 | 显然低估 | — | 合理定价 | 估值偏贵 | 明显高估 |
| 认知优势 | — | 深度研究有差异 | — | 普通了解 | 道听途说 |
| 周期系数 | 底部复苏启动 | 底部未确认 | 中段中性 | 过热 | 明确顶部 |
